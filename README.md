# DBMS-proj
sems 4 dbms semester project

Luxury Rental System
A members-only luxury rental web app built with Flask + PostgreSQL.
Users must sign up/login to access products, while admins manage inventory and rentals.

Features

User signup & login (session-based)
Members-only access
Browse and book luxury products
Admin manages products, bookings, rentals
Reviews, payments, and penalties system



Tech Stack
Flask (Python)
PostgreSQL (pgAdmin)
HTML/CSS (Jinja templates)
psycopg2


Database
Main tables:
users, roles, products, inventory, bookings, rentals, returns, payments, penalties, reviews, categories, brands
